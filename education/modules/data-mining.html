Learning Outcomes

  • Build reproducible pipelines for cleaning, encoding, scaling, and modelling.
  • Handle class imbalance with re-sampling (e.g., SMOTE) and thresholding.
  • Apply clustering (k-means/DBSCAN) and interpret cluster quality.
  • Use ROC-AUC, PR-AUC, and confusion matrices to evaluate models properly.
  • Communicate results with clear plots and short, reproducible notebooks.

Indicative Topics

Data preparation

Missing values, outliers, encoding categoricals, scaling/normalising, train/val/test hygiene.

Imbalanced learning

SMOTE/SMOTE-NC, class weights, threshold moving, PR-AUC focus for rare positives.

Clustering & similarity

k-means (elbow/silhouette), DBSCAN (ε, minPts), distance metrics and feature scaling.

Rules & association

Frequent itemsets (Apriori), support, confidence, lift; pitfalls and spurious rules.

Model evaluation

Stratified k-fold CV, leakage prevention, calibration curves, learning curves.

Quick Reference — Binary Classification (Imbalanced)

Pipeline skeleton

from sklearn.compose import ColumnTransformer
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import OneHotEncoder, StandardScaler
from sklearn.metrics import average_precision_score, classification_report
from imblearn.pipeline import Pipeline as ImbPipeline
from imblearn.over_sampling import SMOTE
from sklearn.linear_model import LogisticRegression

pre = ColumnTransformer([
  ("num", StandardScaler(), num_cols),
  ("cat", OneHotEncoder(handle_unknown="ignore"), cat_cols),
])

pipe = ImbPipeline(steps=[
  ("prep", pre),
  ("smote", SMOTE()),
  ("clf", LogisticRegression(max_iter=500))
])

Use PR-AUC / average precision for rare positive classes.

Clustering cues

# k-means: pick k with elbow/silhouette
# DBSCAN: scale features; tune eps via k-distance plot
# Validate clusters with silhouette and domain sense

Always scale numeric features before distance-based clustering.

Key Takeaways

  • Clean splits and leakage prevention are non-negotiable for valid results.
  • For rare events, PR-AUC is often more informative than ROC-AUC.
  • Resampling and class weights complement each other—validate both.
  • Clustering is exploratory: pair metrics with domain interpretation.
