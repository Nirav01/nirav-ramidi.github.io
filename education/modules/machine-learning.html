Quick Reference

Regularised risk

min_w  (1/n) Σ L(y_i, f_w(x_i)) + λ ||w||ₚ
p = 2 → Ridge,  p = 1 → Lasso

λ controls bias–variance trade-off; scale features before fitting.

Confusion matrix (binary)

Precision = TP / (TP + FP)
Recall    = TP / (TP + FN)
F1        = 2 · (P·R) / (P + R)

Prefer macro-averages for class imbalance.

Lab Snippet — scikit-learn Pipeline

from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.model_selection import StratifiedKFold, GridSearchCV
from sklearn.svm import SVC

pipe = Pipeline([
  ("scale", StandardScaler()),
  ("clf", SVC(kernel="rbf"))
])

param_grid = {
  "clf__C": [1e-2, 1e-1, 1, 10, 100],
  "clf__gamma": [1e-4, 1e-3, 1e-2, 1e-1, 1]
}

cv = StratifiedKFold(n_splits=5, shuffle=True, random_state=42)
gs = GridSearchCV(pipe, param_grid, cv=cv, scoring="f1_macro", n_jobs=-1)
gs.fit(X_train, y_train)

Key Takeaways

  • Clean splits and leakage prevention matter more than fancy models.
  • Metrics must match the business goal; PR-AUC can beat ROC-AUC under heavy imbalance.
  • Scaling + regularisation stabilise optimisation and improve generalisation.
  • Simple baselines (logistic regression) are strong and interpretable; use them first.
