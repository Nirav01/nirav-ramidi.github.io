Quick Reference

Hello + compile/run

#include <stdio.h>
int main(void) {
  printf("Hello, C!\n");
  return 0;
}
/* Compile & run:
   gcc -Wall -Wextra -O2 hello.c -o hello
   ./hello
*/

Enable warnings (-Wall -Wextra) to catch common mistakes early.

Input & loops

#include <stdio.h>
int main(void){
  int n, sum = 0;
  if (scanf("%d", &n) != 1) return 1;
  for (int i=1; i<=n; i++) sum += i;
  printf("sum = %d\n", sum);
  return 0;
}

Always check scanf returns to avoid using uninitialised values.

Functions

int max(int a, int b) { return (a > b) ? a : b; }
int main(void){ printf("%d\n", max(3,7)); }

Prefer small functions; name them after what they return/do.

Arrays & strings

#include <string.h>
int main(void){
  int a[5] = {1,2,3,4,5};
  char name[16] = "Nirav";
  printf("%zu\n", strlen(name));  // 5
  return 0;
}

Mind bounds: arrays are fixed-size; C strings are NUL-terminated.

Tooling & Debugging

# Makefile (mini)
CC=gcc
CFLAGS=-Wall -Wextra -O2 -g
all: app
app: main.o util.o
	$(CC) $(CFLAGS) -o $@ $^
clean:
	rm -f *.o app

# GDB (basics)
gdb ./app
(gdb) break main
(gdb) run
(gdb) next / step
(gdb) print n
(gdb) backtrace

Compile with -g for debugging symbols; use valgrind in later modules to find memory issues.
