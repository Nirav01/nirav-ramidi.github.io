Astronomy Classifier (SDSS)

Multi-class classification of SDSS objects (star / galaxy / quasar) using SVM, kNN, and Decision Trees. Evaluated with ROC-AUC (OvR), macro-F1, and stratified cross-validation.

Dataset & Problem

The task is to classify astronomical objects from the Sloan Digital Sky Survey (SDSS) into three classes: star, galaxy, and quasar (QSO). Inputs include photometric magnitudes (e.g., u, g, r, i, z), colours (magnitude differences), redshift (if available), and quality flags. The dataset is moderately imbalanced.

  • Split strategy: stratified train/test with a validation fold inside training.
  • Targets: 3 classes with one-vs-rest (OvR) evaluation for ROC-AUC.
  • Goal: robust macro-F1 and clear class separation in ROC curves.

Preprocessing & Features

Cleaning

  • Removed records with invalid/flagged magnitudes; imputed rare missing values.
  • Winsorised extreme outliers in colours to reduce tail influence.

Scaling

  • StandardScaler on numeric features (required for SVM/kNN).

Feature Engineering

  • Constructed colour indices (e.g., u-g, g-r, r-i, i-z).
  • Added interaction terms and simple polynomial features for non-linear boundaries.
  • Compared filter methods (mutual information) with RFE wrappers.

Models & Tuning

  • Baselines: Majority class; Logistic Regression (multinomial).
  • kNN: grid over k ∈ {5, 7, 9, 11, 15, 21}, distance metrics, and weighting.
  • SVM (RBF): grid over C and γ; class weights to handle imbalance.
  • Decision Tree: tuned max_depth, min_samples_split, and ccp_alpha (cost-complexity pruning).

All models were evaluated with 5-fold stratified CV on the training set; the selected model was then assessed on the held-out test set.

Evaluation

  • Metrics: macro-F1, per-class precision/recall/F1, micro-F1, and ROC-AUC (OvR).
  • Diagnostics: confusion matrix, learning curves, and calibration curves (optional).

Results

  • SVM (RBF) provided the best macro-F1 on the test set; kNN was competitive with careful scaling and k choice.
  • Decision Trees were most interpretable; pruning reduced overfitting with a small trade-off in metrics.
  • Feature selection modestly improved generalisation; colour indices were among the most informative features.

Lessons & Next Steps

  • Proper scaling is critical for distance-based and kernel methods.
  • Class weighting and careful validation mitigate moderate imbalance.
  • Next: try Gradient Boosting / XGBoost, probability calibration (Platt/Isotonic), and SHAP for explanations.
